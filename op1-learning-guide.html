Chapter 1 — Getting Started

Setup & your first sound

Before making beats, get your OP-1 powered up and playing audio. This takes less than 2 minutes.

Quick start steps
Slide the power switch (next to the USB port on the right side) toward you to power on
Turn up volume using the white knob on the left side of the screen (clockwise = louder)
Press the DRUM key to enter Drum Mode
Press any key 1–8 (sound selection keys) to choose a drum kit
Hit any key on the musical keyboard — you should hear a drum sound!
Essential key layout

Hold to access secondary functions on any key
HELP
Shows battery level + contextual help for current mode
★
Pro tip: Data saves automatically on the OP-1 — you never need to manually save. Everything is written to flash memory as you go.
i
To check battery level: press HELP. The LEDs light up to show the level. Five lit LEDs = full charge.

Understanding the drum layout
01
The 8-slot drum sampler
How drum sounds are organized
▾

Each drum kit has 8 sounds mapped across the keyboard. Sounds 1–8 (the selection keys) choose the kit, while the musical keyboard lets you trigger individual drums.

i
The keyboard is laid out like a standard drum machine: low keys = kick/bass, mid keys = snare/clap, high keys = hats/cymbals.
02
Dynamic envelope per drum
Shape attack, decay, sustain, release
▾

Each drum slot has its own envelope. In Drum Mode, turn the 4 white encoders (knobs) to adjust Attack, Decay, Sustain, and Release for the selected sound.

Green knob
Attack — how fast the sound starts
White knob 2
Decay — how quickly it falls off
White knob 3
Sustain — level while key is held
White knob 4
Release — tail after key is released
03
Importing your own drum sounds
Customize kits via disk mode or sampling
▾

You can replace any drum sound with your own samples. Connect via USB, enable Disk Mode, and drag .aif files into the drum folder. Or use the built-in mic to sample anything live.

★
Handbook tip: You can create a whole drum kit from FM radio waves — tune to a station, then sample individual hits using the MIC/INPUT key in drum mode.

Overdub layers on top
Add bass, synths, and more without erasing
▾

Overdubbing lets you add new sounds on top of existing recordings on the same track. You can also record to a new track (T2, T3, T4) for cleaner separation.

!
Recording to a track that already has audio will overdub — not replace — unless you erase the section first. Use tape editing (LIFT tool) to remove sections.
03
Tape speed tricks
Pitch-shifting and lo-fi textures
▾

Changing the tape speed after recording is a powerful creative trick. Slowing it down pitches everything down and creates a lo-fi, spacious feel. Speeding it up creates tight, pitched-up sounds.

★
Handbook trick: Record a beat at double speed, then slow the tape to half — instant lo-fi, warped drum sound. This technique is mentioned in the OP-1 guide as a "tape trick."
Chapter 4 — Sequencers

Automating your beats

The OP-1 has 6 built-in sequencer types. Each one changes how rhythm and note data flows — from classic step sequencing to chance-based patterns.

The 6 sequencers
All sequencers
Best for beats
END

Endless sequencer

Classic step sequencer — up to 128 steps. Set playback direction, speed, and trigger patterns. The workhorse for programming any repeating beat or melody.

PAT

Pattern sequencer

Program a fixed-length pattern. Assign notes, rests, and ties. Great for drum patterns and melodic loops you want to repeat exactly.

TOM

Tombola sequencer

A ball drops through a grid of pegs, triggering notes as it falls. Semi-random and generative — perfect for organic, evolving percussion and arpeggios.

FIN

Finger sequencer

Each note key holds a different pattern. Two patterns interact when both keys are pressed. Ideal for assigning your beat to one key and fills to others.

SKT

Sketch sequencer

Draw a path directly on screen with the encoders. The path controls pitch and timing. Experimental and expressive — great for glitchy, drawn patterns.

ARP

Arpeggio sequencer

Classic arpeggiator. Press a chord and the arpeggiator breaks it into a sequence. Different modes change the note order and octave range.

i
For building beats, these 3 sequencers work best in Drum Mode. Press SEQ while in Drum Mode to access them.
END

Endless — for drum programming

Program each drum hit across up to 128 steps. Set different speeds per sound for polyrhythmic patterns. This is your go-to drum sequencer.

FIN

Finger — for beats with fills

Assign your main beat to one key, drum fills to others. When two keys are pressed together the patterns interact using a set rule — great for live performance.

TOM

Tombola — for organic percussion

The semi-random ball-drop mechanism creates evolving, never-quite-the-same patterns. Use on a percussion layer over a steady kick/snare for movement.

▾

The LFO (Low Frequency Oscillator) modulates any parameter over time — pitch, filter cutoff, volume. Route it to a hi-hat's volume for a sidechain-style pumping effect.

▾

★

Chapter 7 — Pro Tips

Handbook tricks & workflows

These techniques are pulled directly from the official OP-1 documentation and the OP-School guide series.

From the official handbook
★
Drum kit from FM radio: Tune to a static-heavy station, then sample individual bursts directly into drum slots. Creates unique, organic-feeling noise percussion that nobody else has.
★
Singing birds sound: Use FM engine, short attack/decay with long release, Spring reverb effect, Tombola sequencer. A classic exercise from the OP-1 handbook that demonstrates how sequencer + engine combination creates character.
★
Random parameter trick: In Synth mode, press the RANDOM parameter option to randomize all values in the current module at once. Record the happy accidents before touching anything.
i
COM mode: Connect two OP-1s via a 3.5mm cable and use COM mode to send MIDI note data between them. Build multi-device live setups with one OP-1 clocking another.
i
Ableton Live integration: Use OP-1 Mode or MIDI Controller mode to send MIDI to Live. The official handbook includes specific key assignment mappings for Live control.
!
Album render: When you're done, use Album Mode to bounce your 4 tracks to a stereo file. Connect to a computer in Disk Mode to export the .aif file. This is your finished track.
Beat progression roadmap
①

Simple 4-on-the-floor

Kick on every beat, snare on 2 and 4, hi-hat in 8ths. Record live in Drum Mode to Tape Track 1.

②

Add syncopation

Use the Pattern sequencer to move the kick slightly off the grid. Add ghost notes to the snare.

③

Layer melody

Record a bass line on T2 with FM engine. Add a lead on T3 using the Arpeggio sequencer.

④

Add live variation

Use the Finger sequencer to create fills triggered by key combos. Switch between beats live.

⑤

Texture & effects

Apply LFO modulation, add the Tombola on a percussion layer, and use tape speed to create a lo-fi finish.

⑥

Render the album

Switch to Album Mode and bounce your 4 tracks to stereo. Export via Disk Mode.
